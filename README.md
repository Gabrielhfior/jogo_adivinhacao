# 🔢 Jogo da Adivinhação (Python)

Um jogo simples de terminal onde você precisa adivinhar o número secreto gerado pelo computador.

---

## 📌 Sobre o projeto

Este projeto foi criado para praticar lógica de programação em Python, trabalhando principalmente com:

- Condicionais (`if / elif / else`)
- Laços de repetição (`while`)
- Geração de números aleatórios
- Controle de tentativas do jogador

---

## 🎮 Como funciona

- O computador escolhe um número aleatório entre **1 e 100**
- O jogador tenta adivinhar o número
- A cada tentativa, o jogo informa se o chute foi:
  - Muito alto 📈
  - Muito baixo 📉
- O jogo continua até o jogador acertar ou digitar `200` para sair

### 🏆 Objetivo:
Adivinhar o número no menor número de tentativas possível
